Featured Project

SASHIMI

Spatial Analysis for Segmented Histopathology Images using Machine Intelligence

Web Implementation Open Poster (PDF)

Overview

SASHIMI is a reproducible analytics framework for AI-segmented histopathology images. The project processes and validates biomedical data at scale, constructs spatial feature tables, and evaluates clinical relevance through survival analysis.

What Was Implemented

The pipeline automates data processing in Python and R, runs batch-level quality checks, and benchmarks methods for stability and sensitivity across cohorts and parameter settings. Downstream statistical modeling includes Kaplan-Meier estimation and Cox proportional hazards analysis, producing interpretable summaries such as hazard ratios and log-rank statistics.
