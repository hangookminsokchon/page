# Yoolkyu Park CV Website (Multi-Page)

Static multi-page CV site that runs without a backend.

## Pages
- `index.html`: home (intro, quick education, SASHIMI intro, honors)
- `education.html`: education details
- `sashimi.html`: project details, poster preview, implementation link
- `experience.html`: relevant experience and awards
- `publications.html`: publication details
- `skills.html`: technical skills
- `contact.html`: contact details and resume link

## Assets used
- Profile image: `image/profilepic.jpeg`
- SASHIMI banner: `image/SASHIMI_banner.png`
- Resume: `figures/Yoolkyu_Park_Resume_30 (5).pdf`
- Poster: `figures/ISBA_SASHIMI_Yoolkyu_Park.pdf`

## Run locally
Open `index.html` directly in a browser.

## Publish free
Push to GitHub and enable GitHub Pages for this repo.
