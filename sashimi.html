SASHIMI

Spatial Analysis for Segmented Histopathology Images using Machine Intelligence

Web Implementation Open Poster (PDF)

Overview

SASHIMI is a reproducible analytical framework for AI-segmented histopathology images. The platform processes segmentation-derived cell coordinates as marked point pattern data and systematically constructs spatial feature tables that quantify cellular organization within the tumor microenvironment (TME).

It integrates spatial statistics, topological data analysis, and areal similarity indices to provide both scalar feature outputs and functional visualizations of intercellular spatial interactions. Designed as a user-friendly web application, SASHIMI enables real-time exploratory spatial analysis without requiring extensive computational infrastructure or external package installation.

"Spatial Analysis for AI-segmented Histopathology Images: Methods and Implementation"

Yoolkyu Park, Fangjiang Wu, Xin Feng, Shengjie Yang, Elizabeth H. Wang, Bo Yao, Chul Moon, Guanghua Xiao, and Qiwei Li

In this paper, we systematically organize and implement 27 spatial descriptors, including classical spatial summary statistics, areal similarity indices, and topological features for marked point pattern data derived from AI-segmented histopathology images.

We further conduct survival analyses on two real-world datasets (NLST and OPMD) using functional Cox proportional hazards (FCoxPH) models to investigate the relationship between spatial architectural features and patient survival outcomes.
